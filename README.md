# Troopr Assignement 
This is a simple crud based project in NodeJS + MongoDb+ ReactJs.
## Clone Repository
```
git clone <repo_url>

```
## Steps to up Backend Server-

```
using Docker
$ docker-compose up -d
$ docker logs -f api
$ docker-compose restart api
```
```
using npm command
$ cd /backend
$ npm i
$ npm start
```
## Steps to up Frontend Server-
```using npm command
$ cd /frontend
$ npm i
$ npm start
```

## Endpoints


| Method     |  URL |
